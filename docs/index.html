ONGOING PROJECT DEVELOPMENT

Conjclassts: Python Package for Enumeration of Conjugacy classes in Transformation Semigroup

In Development

Software Development Project (Builder: Clinton Oluranran Kayoh)

Developing a comprehensive Python package designed to facilitate advanced computational analysis of transformation semigroups and their algebraic properties. Conjclassts aims to become an essential tool for researchers in algebraic combinatorics and computational algebra.

Package Features:

  • Enumeration of Conjugacy classes in transformation semigroups
  • Sub-semigroup identification and classification
  • Conjugacy class computation and analysis
  • Nilpotency detection for semigroups and conjugacy classes
  • Idempotency analysis for semigroups and their conjugacy classes
  • Efficient algorithms for large-scale computations
  • User-friendly API for mathematical research applications

ONGOING COLLABORATIVE RESEARCH

Non-Starlike and Quasi-Starlike Transformation Semigroups

In Progress

Collaboration with Ugbene Lab

An in-depth investigation into the algebraic and combinatorial properties of transformation semigroups, exploring their structural characteristics, graphical representations, and various subsemigroup classifications.

Research Focus:

  • Algebraic properties of non-starlike and quasi-starlike transformation semigroups
  • Combinatorial properties and enumeration techniques
  • Digraph representations and visualization methods
  • Conjugacy class analysis and classification
  • Subsemigroups: order preserving, contraction, order decreasing
  • Order preserving or reversing transformation properties

Collaborators: Rhoda Osanakpa, Clinton Kayoh, Salami Matthew, Emunefe Friday, and the Ugbene Lab

ONGOING COLLABORATIVE RESEARCH

Mixing Time of Markov Chains on Graphs and Network Structures

In Progress

Collaboration with Ugbene Lab

Investigating the mixing time behavior of Markov chains across various graph topologies and network architectures, with emphasis on practical applications in computational systems, social networks, and biological processes.

Research Focus:

  • Mixing time analysis for different graph structures (regular, random, scale-free)
  • Network topology impact on convergence rates
  • Spectral methods and eigenvalue approaches
  • Applications in random walks and diffusion processes
  • Real-world implementations in communication networks
  • Computational algorithms for mixing time estimation

Collaborators: Elohor Imelda Agbedo, Clinton Oluranran Kayoh and the Ugbene Lab
